WHAT MAKES THE CHANGE
From our investigation(?) we saw 3 possible options to make the space change. TIMED RELAY. The advantages to using this kind of change, is it's predicatable and allows someone to use the space without direct or indirect participants. Negatives of this kind of change is that it's predicatable and we're not 100% sure that its still possible to call it an interface, since what people do has no impact to how the space changes. DIRECT USER INTERACTION. Advantages of this kind of change, is it lets the user understand how the space is changing. But we wonder if it will add to a sense of comradery between primary and secondary users. INDIRECT USER INTERACTION. Advantages of this is that it adds mystetry to how the space changes, it isolates the user from the actions that influence the space they occupy. Disadvantages to this kind of interaction is that the space won't react the same every time and the primary user isn't guaranteed a dynamic space of isolation and agoraphobia

HOW DO WE WANT TO OCCUPY SPACE
Depending which design we design to move forward with, there are two possibilties to continue with, either within an enclosed space or an open space. We could inflate the space within an enclosed space, which could allow for a more private experience, with the secondary user outside of the space. This could also possibly lower the risk of blow out as we processed into fabrication and testing. This could also remove the need for an enclosing inflatable if we decide to continue with several volumes within a single volume, the fixed enclosure acting as the single volume. If we continue to design with an open space in mind, this opens up the possibilities of spaces we could occupy throughout Storrs, through the primary location is the Saloon, and the rest of the university. If open the inflatable becomes a spectacle to be observed, it changes from an experience from individual to collective. Both are conducive to the Covid experience, since we expereinced Covid as individuals and as a collective. With "clear" polyethylene material, it still obstructs the views from both inside and outside.

A problem that has been nagging the back of the old noggin' is how does this project contribute to the research field of inflatables? This project was sparked by an emotion, and it feels unfair to replace that passion with rational ideas with the intent to contribute something to scientific reseach. Art exhibits still have the ability to contribute to science without removing itself from the passion that started and drove a project
